Conversion of T1 Nifti image to 3D printable STL file
Line Kruse
Created 10.05.23
Step-by-step guide for converting structural fMRI T1-weighted nifti images into STL format which can be printed on 3D printer.
T1 images should be in Nitfi (.nii) format.
Using FreeSurfer and Meshlab.
Based on guides from:
https://kathleenhupfeld.com/how-to-3d-print-someones-brain-or-how-to-convert-a-t1-weighted-mri-scan-to-a-printable-stl-file/
https://layerfmri.com/2018/07/25/3d-printing-nii-data/
Download and install FreeSurfer
Download FreeSurfer
rel7downloads - Free Surfer Wiki
https://surfer.nmr.mgh.harvard.edu/fswiki/rel7downloads
Unpack/Install FreeSurfer
You laptop might refuse to open it saying it’s from unidentified developers. In this case go to Security and Safety settings and allow it to open.
Setup FreeSurfer environment
In terminal, setup a FreeSurfer environment with the following three lines of code.
First line (setting FREESURFER_HOME) should point to the directory of your installed FreeSurfer folder.
JavaScript
Copy
export FREESURFER_HOME=/Applications/FreeSurfer/ export SUBJECTS_DIR=$FREESURFER_HOME/subjects source $FREESURFER_HOME/SetupFreeSurfer.sh
​
Should give an output that looks something like this…
Place your T1 Nifti image in the subjects folder
The nifti image should be in /Applications/freesurfer/subjects
Preprocess T1 image with FreeSurfer
In this step FreeSurfer will produce cortical surface maps that you can convert into a STL file for 3D printing.
NB!
Make sure you have read and write rights to the SUBJECTS_DIR folder.
Make sure you have a valid freesurfer license and it is stored in the /Applications/freesurfer/ folder. If you dont have one, register and get one for free here: https://surfer.nmr.mgh.harvard.edu/registration.html
Run code:
This job takes about 5 hours to run per image
JavaScript
Copy
recon-all -all -i /Applications/FreeSurfer/subjects/filename.nii -s output_folder
​
Set correct path to FreeSurfer/subjects folder (according to your laptop)
Set correct name of your nifti file (filename.nii)
Specify the name of the output folder follwing the -s command
Next step…
JavaScript
Copy
mris_convert /Applications/FreeSurfer/Subjects/output_folder/surf/rh.pial rh.stl Saving rh.stl as a surface
​
Then the left hemisphere
JavaScript
Copy
mris_convert /Applications/FreeSurfer/subjects/output_folder/surf/lh.pial lh.stl Saving lh.stl as a surface
​
Open your FreeSurfer file in Meshlab
Download Meshlab from:
https://sourceforge.net/projects/meshlab/files/meshlab/MeshLab v1.3.3/
Open meshlab and import left hemisphere file:
JavaScript
Copy
File > Import mesh > /Applications/FreeSurfer/subjects/output_file/lh.stl
​
A pop-up window will ask if you want to unify duplicated vertices - Press “Ok”.
Import right hemisphere in the same way:
JavaScript
Copy
File > Import mesh > /Applications/FreeSurfer/subjects/output_file/rh.stl
​
You shold now have a 3D object of the entire brain:
Edit the 3D object with a few filters
First, flatten visible layers
JavaScript
Copy
Filters > Mesh Layer > Flatten Visible Layers > [keep all defaults checked] > Apply
​
Cut out holes and merge overlapping objects
JavaScript
Copy
Filters > Remeshing Simplification and Reconstruction > Quadric Edge Collapse Decimation
​
The number of “Faces” should be set between 200.000-270.000
Smooth the model
JavaScript
Copy
Filters > Smoothing, Fairing and Deformation > HC Laplacian Smooth
​
Now you have a nice smooth 3D brain.
Note that the 3D object now is in its true size, i.e., the true size of the actual brain. If you want it to be smaller/larger, you can rescale it.
The original size has X, Y and Z scale factors of 1.
If you, for example, want it to be half the size, you can set X, Y and Z scale factors to 0.5.
JavaScript
Copy
Filters > Normals, Curvatures, and Orientation > Transform: Scale
​
Export 3D mesh as STL file
JavaScript
Copy
File > Export mesh [> set output to STL > name file > used all defaults]
​
And there you go! You are ready to print! ​
